## What's This All About?

MeloDoro transforms your boring study routine into a delightfully visual experience. Watch a watermelon slice fill up as you focus, track your daily goals with an animated watermelon glass, and build study streaks that would make even the most dedicated fruit farmer jealous.

## Getting Started

1. **Clone this repository**

   ```bash
   git clone https://github.com/yourusername/watermelon_timer.git
   cd watermelon_timer
   ```

2. **Install dependencies** (Flutter required - if you don't have it, [get it here](https://flutter.dev/docs/get-started/install))

   ```bash
   flutter pub get
   ```

3. **Run the app**

   ```bash
   flutter run
   ```

4. **Start studying** (the hard part, but we believe in you!)
